学习Python过程中，我在网络上收集的资料。

教材

  • The Hitchhiker’s Guide to Python! 包罗万象的一个指南，里面有你所需要的获取的各种Python资源。

  • A Byte of Python for Python 我个人认为是简明的Python入门教程。

  • Think Python: How to Think Like a Computer Scientist 除了教你入门Python，还介绍了一些计算机科学方面的内容。

  • Dive Into Python 3 稍有点难度的Python入门教程，同时也很有趣。

  • Learn Python the Hard Way 反直觉的Python入门教程，坚持下来之后学习效果会非常棒。

  • Automate the Boring Stuff with Python 用很多实用的例子让我见识到了Python在消除重复劳动方面的功能。

实践

  • PYTHON CHALLENGE。 除了需要Python技能外，解决问题时还需要一点脑筋急转弯。这让整个解决过程显得特别有趣。The Python Challenge wiki展示了很多解决方案，值得研究。另一个网站也提供了自己的解决方案，Python Challenge - Solutions。

  • Python 练习册，每天一个小程序 这些小程序能够帮助我们熟悉基本语法，并通过完成一些小项目来增加学习的信心和乐趣。里面的链接稍显陈旧，需要自己搜索新的资料。
