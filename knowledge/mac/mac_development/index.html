Homebrew

Homebrew. The Missing Package Manager for macOS (or Linux)

Homebrew类似于Linux上的apt和rpm。我在Mac上使用之后，觉得非常方便。

按照首页说明安装之后，它默认安装在/usb/local/bin下面。

where brew

/usr/local/bin/brew

之后，就可以使用Homebrew来管理各种包了。

Homebrew默认将包都安装在/usr/local/Cellar下面。

ls -al /usr/local/Cellar

total 0
drwxrwxr-x@ 120 zhoulinux  admin  3840 Aug 11 09:54 .
drwxr-xr-x   20 root       wheel   640 Jul 22 18:17 ..
-rw-r--r--    1 zhoulinux  admin     0 Oct 16  2019 .keepme
drwxr-xr-x    3 zhoulinux  admin    96 Jul  7 09:33 apr
drwxr-xr-x    3 zhoulinux  admin    96 Jul  7 09:33 apr-util
drwxr-xr-x    3 zhoulinux  staff    96 May 17  2020 aria2
...

这里的/usr/local是通过变量prefix来指定的。

brew --prefix

/usr/local

Homebrew下载过程中的缓存文件存放路径，是通过变量cache来指定的。

brew --cache

/Users/zhoulinux/Library/Caches/Homebrew

安装完毕后，Homebrew将这些可执行程序链接至/usr/bin中，例如Pyhon。

ls -al py*

-rwxr-xr-x  1 zhoulinux  admin  228 Apr 23  2018 py.test
lrwxr-xr-x  1 zhoulinux  admin   37 Jul  6 11:44 pydoc3 -> ../Cellar/python@3.9/3.9.6/bin/pydoc3
lrwxr-xr-x  1 zhoulinux  admin   39 Jul  6 11:44 pydoc3.9 -> ../Cellar/python@3.9/3.9.6/bin/pydoc3.9
lrwxr-xr-x  1 zhoulinux  admin   38 Jul  6 11:44 python3 -> ../Cellar/python@3.9/3.9.6/bin/python3
lrwxr-xr-x  1 zhoulinux  admin   45 Jul  6 11:44 python3-config -> ../Cellar/python@3.9/3.9.6/bin/python3-config
lrwxr-xr-x  1 zhoulinux  admin   40 Jul  6 11:44 python3.9 -> ../Cellar/python@3.9/3.9.6/bin/python3.9
lrwxr-xr-x  1 zhoulinux  admin   47 Jul  6 11:44 python3.9-config -> ../Cellar/python@3.9/3.9.6/bin/python3.9-config

除了安装各种包之外，我们还可以使用cask来安装经常使用的图形软件，例如Firefox。

brew install --cask firefox

这些图形软件会直接安装在/Applications下面。
